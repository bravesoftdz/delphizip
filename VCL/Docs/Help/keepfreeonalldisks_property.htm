KeepFreeOnAllDisks property

Applies to

TZipMaster component

Declaration

property KeepFreeOnAllDisks: Cardinal;

Description

Holds a value in bytes and will use this value to keep bytes free on all disks of a set.

For example to leave slack space CDs.
