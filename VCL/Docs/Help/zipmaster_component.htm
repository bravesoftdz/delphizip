TZipMaster - Version 1.91

 

Main Unit

ZipMstr.pas

TZipMaster is a control designed to simplify the handling of Zip compressed files.

Description:

TZipMaster is a non-visual VCL component for managing zip files with some functions being a wrapper for a combined version of freeware Zip and Unzip dlls.

At run time for zip and unzip operations, DelZip190.dll must be present on the hard disk where the application can find it - in ?:\WINDOWS\SYSTEM32 or else in your application directory, or a directory in the PATH.

If the DLL is statically linked to the component then the IDE must be able to locate and load the DLL .

It is also possible for an application to access a compressed version of the DLL stored in the it's resources (dynamic binding only).

The DLL is now fully re-entrant so multiple components (or applications) can share a single DLL .

This DLL is based on the InfoZip Official Freeware Zip/Unzip source code, but they are NOT equivalent to InfoZip's DLL's. We have modified the InfoZip source code to enhance their ease-of-use, power, and flexibility for use with Delphi and C++ Builder.  Please do NOT contact InfoZip for issues regarding this port.

TZipMaster can encrypt and decrypt files through the Password property.

Many operations do not require the dll.

The OnMessage event fires when the  DLL reports status. The Trace and Verbose properties control the amount of information written to the ErrMessage property.
