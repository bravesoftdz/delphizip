VersStr function

Unit

ZMUtils

Declaration

function VersStr(vers: Integer; Comma: Boolean = False): String

Arguments
vers: Integer
A build number
Comma: Boolean = False
controls using dot or comma as separator
Results
  • Returns the resulting String
Description

Converts our version value to a readable string.

If Comma is true the format is '%d,%d,%d,%d'

if false the format is '%d.%d.%d.%4.4d'
