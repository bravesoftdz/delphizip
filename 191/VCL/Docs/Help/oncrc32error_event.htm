OnCRC32Error event

Applies to

TZipMaster component

Declaration

property OnCRC32Error: TZMCRC32ErrorEvent;

Type
  TZMCRC32ErrorEvent = Procedure( Sender: TObject; const ForFile: TZMString; FoundCRC, ExpectedCRC: LongWord; var DoExtract: Boolean ) of object;

Arguments
Sender: TObject
The component firing the event
const ForFile: TZMString
The name of the file
FoundCRC: Longword
The calculated CRC
ExpectedCRC: Longword
The required CRC
var DoExtract: Boolean
Set True to force extraction
Description

Occurs when during an Extract a CRC error is found in the file ForFile.

You can choose to skip the extraction of the file by setting DoExtract to False. The default for DoExtract is True because that was the case in previous versions of ZipMaster where you could not choose.
