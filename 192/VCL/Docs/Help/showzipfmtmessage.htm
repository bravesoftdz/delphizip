ShowZipFmtMessage method

  • ErrMessage
  • OnMessage
  • ShowZipMessage
Applies to

TZipMaster component

Declaration

procedure ShowZipFmtMessage(Id: Integer; const Args: array of const);

Arguments
Id: Integer
a message identifier
const Args: array of const
the required arguments for that message
Description

The ShowZipMessage method reads a message identified by Ident from the program resource table, formats it and displays it.

All component messages, errors and warnings are as of version 1.5 contained in a resource message table.

To make it easier to also use them in your own program this function is included.

It reads the message with identifier 'Ident' from the resource table, formats using 'Args' and displays the message.

If the identifier is not present in the selected language the default message in US English will be displayed.

Reasons why the identifier may not be present normally :

  • The resource file is not included in your project: include the ZMRes_str.res, created by ZipResMaker, into your project.
  • Your lanugage is not included in the resource file : with ZipResMaker include your language and rebuild the resource file, remember to rebuild your project as well.
  • Your language file 'ZipMsg??.rc' does not include a translation for this item : Add the translation to the 'ZipMsg??.rc' file, rebuild the resource with ZipResMaker and rebuild your project.
