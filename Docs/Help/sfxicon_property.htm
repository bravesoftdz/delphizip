Applies to

TZipMaster component

Declaration

property SFXIcon: TIcon;

Description

Used to set a custom icon for the Self Extracting Archive. This is the picture you see in the explorer and on your desktop.

Because the original size of the icon in SFX stub has a fixed size, the size and number of colors used must be exactly the same this means: 32 x 32 pixels and 16 colors.

If you supply an icon which is not exactly the same you will get error CZ_NoCopyIcon when using the ConvertToSFX method.

This property is automatically set when converting (back) from .Exe to .Zip.
