PWideToUTF8 function

Unit

ZMUTF8

Declaration

function PWideToUTF8(const pwstr: PWideChar; len: integer = -1): UTF8String;

function WideToUTF8(const astr: TZMWideString; len: integer = -1): UTF8String;

Arguments
const pwstr: PWideChar
Pointer to wide characters (UTF16)
len: integer
Number of WideChar to convert (<0 - to end of string)
const astr: TZMWideString
WideString (UTF16) to convert
len: integer
Number of WideChar to convert (<0 - to end of string)
Results
  • Returns the resulting UTF8String
Description

Converts a string of WideChar to UTF8.

If len = -1 all of the characters up to the terminating #0 are used.
