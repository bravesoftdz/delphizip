RootDir property

  • Add
  • FSpecArgs
  • FSpecArgsExcl
  • AddOptions
  • Using Relative Paths
Applies to

TZipMaster component

Declaration

property RootDir: String

Description

The RootDir property determines where zipping will start for any filenames or wildcard entries selected in the FSpecArgs property,

except those excluded by the FSpecArgExcl property.

Example
 Example:        
 ZipMaster1.RootDir := 'D:\Program Files\Start';        
 ZipMaster1.FSpecArgs.Add( 'Dir1\MyFile.html' );        
 ZipMaster1.FSpecArgs.Add( '*.h' );        
 ZipMaster1.Add;        
          
 This will add the file "D:\Program Files\Start\Dir1\MyFile.htm" as "Dir1\MyFile.html"        
 and all files "D:\Program Files\Start\*.h" as "*.h" to the ziparchive.
