MinFreeVolumeSize property

Applies to

TZipMaster component

Declaration

property MinFreeVolumeSize: Integer;

Description

This contains a value in bytes of the minimum available size a disk must have before it will be used to write (a part of) the archive.

The default is 65536 bytes.

NOTE: It is not very practical to set this to a smaller value because you will need a lot of disks or get many files.
