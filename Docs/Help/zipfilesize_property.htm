ZipFileSize property

Applies to

TZipMaster component

Declaration

property ZipFileSize: Int64;

Description

ZipFileSize contains the size of entire Zip file (if it's a spanned set, then size of the last chunk).

ZipFileSize is valid after a zip file's contents have been read successfully (which happens when you assign something to ZipFileName).

Run-time only

Read-only
