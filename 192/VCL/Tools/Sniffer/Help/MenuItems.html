Menu Items

When you go over it with the mouse cursor a hint will appear at the bottom left of the main form .

ZipSniffer has 28 Menu items in 6 groups on its main menu :

  1. File Items
    • Save Log : Creates a log file and puts the content of the message buffer in that file. The log file is named dzlogxxx.txt where xxx is a number from 001, if that file is already open then we use 002 and so on.
    • New : Creates a new child form, you can use this form as you like.
    • Open : Opens a file selection dialog, the purpose is to open one of the log files, then it creates a new child form and shows the content of the file in that form.
    • Close : Closes the child form that has focus
    • Save : Not yet implemented
    • Save As: Not yet implemented
    • Exit : Will close ZipSniffer, before closing the tool a Save Log operation is executed.
  2. Edit Items
    • Cut : Cuts the actual selection and saves it to the clipboard, make a selection in one of the child forms first.
    • Copy : Copies the actual selection and puts it on the clipboard, make a selection in one of the child forms first.
    • Paste : Pastes the clipboard content into the selected child form.
  3. Window Items
    • Cascade: cascades the child forms.
    • Tile Horizontally : arranges child forms so that they are all the same size, tiled horizontally.
    • Tile Vertically : arranges child forms so that they are all the same size, tiled vertically.
    • Minimize All : minimizes all of the child forms.
    • Arrange All : Arranges all minimized windows, lines them up to the Left of the bottom of the main form.
  4. Verbosity
    • Default : as set by code
    • Force Off : minimal messages
    • Force Verbose : information messages
    • Force Trace : Information and diagnostics messages
    • Force Full : Noisy, all messages (as helpful as it can be but slow
  5. Logs
    • Folder : set where logs are written
    • Base name : set the name of log files
    • New : forces starting a new log file
    • Close : closes the current file
    • Pause : suspends writing to the current file
  6. Help Items
    • HelpContent : to call this help file
    • About : that displays the version info for this application.
