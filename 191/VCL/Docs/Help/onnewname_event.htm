Applies to

TZipMaster component

Declaration

property OnNewName: TZMNewNameEvent;

type
  TZMNewNameEvent = procedure(Sender: TObject; SeqNo: Integer); of object

Arguments
Sender: TObject
The component firing the event
SeqNo: Integer
The zip directory entry number
Description

The event OnNewName occurs when reading the Zip's archive directory and gives you the opportunity to fill in for example a TListView.

SeqNo is a number from 1 to the number of entries in the zipfile and can be used to reset a ListView when the value is 1 and as sequence number in a TStringGrid.
