Applies to

TZipMaster component

Declaration

function Add: Integer;

Arguments
ZipFileName property
The file to add to
FSpecArgs property
The files to add
FSpecArgExcl property
The files to exclude
AddOptions property
controls selecting and adding files
WriteOptions property
controls how to write the resulting zip
SpanOptions property
Controls how to span the resulting zip
AddStoreSuffixes property
The file extensions of files to store
Results
  • Returns 0 on success otherwise a - Error Code.
  • Sets ErrCode and ErrMessage
  • sets SuccessCnt to the number of files added.
Description

The Add method adds the specified files to the archive.

Adds all files specified in the FSpecArgs property and not excluded in the FSpecArgExcl property into the archive specified by the ZipFileName property.

Files specified by the AddStoreSuffixes set, that are already compressed, will not be compressed again, but will be stored "as is" in the archive. This applies normally to .ARC, .ARJ, .GIF, .LZH, .PNG, .TAZ, .TGZ, .Z, .ZIP and .ZOO files.

If NoSkipping is True all selected files must be successfully processed otherwise the operation will fail.

It is also possible to use relative paths but your own program has to perform a few steps.

If an operation is already in progress it returns ErrCode or BUSY_ERROR.
