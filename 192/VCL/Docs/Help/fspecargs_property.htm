FSpecArgs property

  • Switches
  • FSpecArgsExcl
Applies to

TZipMaster component

Declaration

property FSpecArgs: TStrings;

Description

Stringlist containing all the file specifications used as arguments for the following methods

  • Add
  • AddZippedFiles
  • ChangeFileDetails
  • CopyZippedFiles
  • Delete
  • Extract
  • ForEach
  • MergeZippedFiles

Every entry can contain MS-DOS wildcards in the name.ext only, NOT the path.

If you give filenames without pathnames, or if you use with relative pathnames, the base drive/directory is assumed to be that of the Zipfile.

You can specify either the MS-DOS backslash path symbol, or the one normally used by PKZip (the Unix path separator: /). They are treated exactly the same.

Add and Extract can also specify special switches to override some default options.

NOTE: Before D2009,  if UseUTF8 is True the strings must be UTF8.

Example
 Add entries directly to the FSpecArgs property:        
 ZipMaster1.FSpecArgs.Add( 'C:\AUTOEXEC.BAT' );        
 ZipMaster1.FSpecArgs.Add( '*.EXE' );        
 ZipMaster1.FSpecArgs.Add( 'C:\WINDOWS\*.INI' );        
 And some special cases which can be used when adding to zip archive:        
 ZipMaster1.FSpecArgs.Add( '>C:\Windows\System\*.Dll' ); // Force recusion
 ZipMaster1.FSpecArgs.Add( 'C:\Windows\System\*.Dll /R' ); // Force recusion
 ZipMaster1.FSpecArgs.Add( '|C:\Windows\*.exe' ); // Stop recursion
 ZipMaster1.FSpecArgs.Add( 'C:\Windows\*.exe /r-' ); // Stop recursion
 ZipMaster1.FSpecArgs.Add( 'C:\Bat\Test.txt<MyNewPassword' ); // A new password
 ZipMaster1.FSpecArgs.Add( 'C:\Bat\Test.txt<' ); // Cancel existing password
 ZipMaster1.FSpecArgs.Add( 'C:\Bat\Test.txt /c:2' ); // Set compression level to 2
 ZipMaster1.FSpecArgs.Add( 'C:\Bat\Test.txt /C:0'); // Set compression level to 0 (store)
 ZipMaster1.FSpecArgs.Add( 'C:\*.* /E:*\.dll\*' ); // only exclude *\.dll\*
 ZipMaster1.FSpecArgs.Add( 'C:\*.* /E:|*\asc*' ); // also exclude  *\asc*
 ZipMaster1.FSpecArgs.Add( '>*.* /E:|*.dll|*.dcu /f:D:/Sources' );          
      // recursive all files from new root except *.dll or *.dcu or anything in FSpecArgsExcl        
 ZipMaster1.FSpecArgs.Add( '* /E:*.dll|*.dcu /f:D:\Sources /s- >MyNewPassword' );          
      // nonrecursive files from new root except *.dll or *.dcu, encrypted with new password        
          
 Take the filespecs from a StringList, just assign them all over to ZipMaster1.        
 ZipMaster1.FSpecArgs.Assign( StringList1 );        
          
 Take the filespecs from a ListBox, just assign them all over to ZipMaster1.        
 ZipMaster1.FSpecArgs.Assign( ListBox1.Items );
