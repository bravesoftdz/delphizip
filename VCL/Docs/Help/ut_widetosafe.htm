Unit

ZMUTF8

Declaration

function WideToSafe(const wstr: TZMWideString; ToOEM: boolean): AnsiString;

Arguments
const wstr: TZMWideString
the string to make safe
ToOEM: boolean
  • True - use code page CP_OEMCP
  • False - use codepage CP_ACP (current)
Results
  • Returns an AnsiString with only safe (ASCII) characters)
Description

Convert a TZMWideString to AnsiString escaping unsupported characters.
