AddZippedFiles method

  • MergeZippedFiles
  • Working with SFX
Applies to

TZipMaster component

Declaration

function AddZippedFiles(SrcZipMaster: TCustomZipMaster; merge: TZMMergeOpts): Integer;

Depreciated new applications should use MergeZippedFiles

Arguments
SrcZipMaster: TCustomZipMaster
The ZipMaster holding the files to be copied
merge: TZMMergeOpts

merge controls how duplicates will be handled:

zmoConfirm
Ask the user if the destination may be overwritten when a source file is found inside the destination or performs the OnCopyZipOverwrite event when assigned.
zmoAlwaysAlways overwrite existing files in the destination
zmoNewerOverwrite only when the source file is younger than the destination file
zmoOlderOverwrite only when the source file is older than the destination file
zmoNeverNever overwrite - skip those files, afterwards these files can be found in FSpecArgs from the source.
FSpecArgs and FSpecArgsExcl
the files to copy
Other properties controlling writing zip files
Results
  • Returns 0 on success otherwise a negative value equal to - Error Code.
  • Sets ErrCode and ErrMessage
Description

Adds selected zipped files from another ZipMaster.

Example
 // copy AutoExec.bat and Config.sys to another zip
 Zip1.ZipFileName := 'SourceZip.zip';       // Open the source        
 if Zip1.ErrCode  0 then         // check no error
    handle_the_error!        
 Zip1.FSpecArgs.Clear;        
 Zip1.FSpecArgs.Add('AutoExec.bat');       // select required files
 Zip1.FSpecArgs.Add('Config.sys');        
             
 Zip2:= TZipMaster.Create(self);       // create ZipMaster 2        
 Zip2.ZipFileName := 'DestinationZip.zip';       // set the destination        
 if Zip2.ErrCode  0 then        
    handle_the_error!               
          
 Result := Zip2.AddZippedFiles(Zip1, zmoConfirm);   // copy files to destination
 if Result  0 then        
    handle_the_error!     
(*
File(s) that are not copied because you answered NO on the overwrite question, 
  or in case zmoNever was specified, 
  can be found in the FSpecArgs list from the source afterwards.
*)
