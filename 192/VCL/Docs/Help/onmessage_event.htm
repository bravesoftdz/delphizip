OnMessage event

  • ErrCode
  • Verbose
  • Trace
Applies to

TZipMaster component

Declaration

property OnMessage: TZMMessageEvent;

Type
  TZMMessageEvent = procedure(Sender: TObject; ErrCode: Integer; ErrMsg: TZMString) of object;

Arguments
Sender: TObject
The component firing the event
ErrCode: Integer
The error code - see Error Codes
const ErrMsg: TZMString
It message string
Description

The ErrMsg argument passed by this event will contain the message.

If an error code accompanies the message, it will be in the ErrCode property.

The Verbose and Trace properties have a direct influence on how many OnMessage events you will get.
