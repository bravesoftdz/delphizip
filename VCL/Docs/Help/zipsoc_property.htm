ZipSOC property

Applies to

TZipMaster component

Declaration

property ZipSOC: Int64;

Description

ZipSOC contains the location measured from the start of the zip file of the Start-Of-Central-Dir.

ZipSOC is valid after a zip file's contents have been read successfully (which happens when you assign something to ZipFileName).

Run-time only

Read-only
