Applies to

TZipMaster component

Declaration

property DLL_Build: Integer;

Description

  Returns the build number of the loaded DLL.

     := ( (1.90 * 100000) + (REVISION * 1000) + PATCHLEVEL )

NOTE: if the dll is not already loaded the result will be 0.

Run-time only

Read-only
