Applies to

TZipMaster component

Declaration

function ConvertToSpanSFX(const OutFile: String): Integer;

Returns

Returns 0 on success otherwise an Error Code.

Description

Convert Zip archive to self-extracting .EXE with a detached header (loader) and convert the zip file to a spanned zip.

OutFile is the name of the resulting .EXE file and the base name for the spanned volumes.

This function returns 0 on success or a negative error value on failure Error Code.

IMPORTANT: Before using ConvertSFX, you may want to first set

  • SFXCaption
  • SFXCommandLine
  • SFXDefaultDir
  • SFXIcon
  • SFXMessage
  • SFXOptions
  • SFXOverwriteMode
  • SFXRegFailPath

  • SFXPath property to the full pathname of the SFX stub.
    If you don't set this property, ZipMaster will automatically look for this file in the Current directory, the application directory, the Windows System and Windows directories and if present the directory specified by the DLLDirectory property.
