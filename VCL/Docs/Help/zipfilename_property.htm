ZipFileName property

Applies to

TZipMaster component

Declaration

property ZipFileName: String

Description

ZipFileName contains the Zip archive filename.

Pathname of a Zip archive file.  If the file doesn't already exist, you will only be able to use the Add method.  

It is recommended to use a fully qualified pathname in this property, unless your program can always ensure that a known directory will be the "current" directory.

NOTE: Some filenames are illegal (operating system issue) like Con, Com1, Com2, LPT1-LPT9.

NOTE: When you assign a value to this property, the contents of the zipfile will be read.
