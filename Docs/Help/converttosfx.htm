Applies to

TZipMaster component

Declaration

function ConvertToSFX: Integer;

Results
  • Returns 0 on success otherwise a negative value equal to - Error Code.
  • Sets ErrCode and ErrMessage
Description

Convert Zip archive to self-extracting .EXE. The resulting .EXE is a Win32 program. Uses the same base name and path already used by the Zipfile - only the file extension is changed to "EXE". This is accomplished by pre-pending the SFX code in  front of the Zip file, and then changing its extension.

IMPORTANT: Before using ConvertSFX, you may want to first set

  • SFXCaption
  • SFXCommandLine
  • SFXDefaultDir
  • SFXIcon
  • SFXMessage
  • SFXOptions
  • SFXOverwriteMode
  • SFXRegFailPath

  • SFXPath property to the full pathname of the SFX stub.
    If you don't set this property, ZipMaster will automatically look for this file in the Current directory, the application directory, the Windows System and Windows directories and if present the directory specified by the DLLDirectory property.
