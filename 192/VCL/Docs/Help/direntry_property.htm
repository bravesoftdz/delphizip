DirEntry property

Applies to

TZipMaster component

Declaration

property DirEntry[idx: Integer]: TZMDirEntry;

Description

Returns a read-only copy of the indexed central directory entry

Run-time only

Read-only

Example
 // read filenames        
 for i:=0 to ZipMaster1.Count-1 do        
    Memo1.Lines.Add(ZipMaster1.DirEntry[i].Filename);
