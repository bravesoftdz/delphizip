OnLoadStr event

  • ZipLoadStr
  • Language

Applies to

TZipMaster component

Declaration

var
  OnLoadStr: TZMLoadStrEvent;

type
  TZMLoadStrEvent = procedure(Ident: Integer; var DefStr: String) of object;

Arguments
Ident: Integer
The identifier
var DefStr: String
On entry contains the default string, can be changed to required string
Description

OnLoadStr event occurs when any TZipMaster component needs to load a message or error string from its resource file.

This gives you the chance to replace this with a string from another external resources such as an language dll.

Ident is the identifier of the message or error string required , the result will be returned in the Str parameter

OnLoadStr is a global event which will handle events from all TZipMaster components.

The message identifiers are declared in ZMMsg.pas.
