Applies to

TZipMaster component

Declaration

property SFXPath: String

Description

SFXPath is the path where the binary stub file is stored.

Must include the whole pathname, filename, and extension.

This is used by the ConvertToSFX method or writing/rewriting an SFX file.

If you don't give the location yourself ZipMaster will try to locate the file as following:

  1. Location given by the SFXPath property. (This could be another name too.)
  2. The current directory.
  3. The application directory. (location of current .EXE file.)
  4. The Windows System directory. (where DLL's go.)
  5. Location given by the DLLDirectory property.
  6. In the application's resources.

When searching, the default stub name is used, except when given an explicit name by means of this property.

Set SFXPath to '><' specify that the stub in the resources must be used.

Example
 zip.SFXPath := 'c:\myapp\zmsfu191.bin'; // must use the specified file
 zip.SFXPath := 'zmsfu191.bin';          // search for the specified stub in default folders
 zip.SFXPath := '.\bin\zmsfu191.bin';    // search for specified stub in the subfolder 'bin\'
 zip.SFXPath := '.\bin\';                // search for default stub in the subfolder 'bin\'
 zip.SFXPath := '><';                    // use the stub in the attached resources
