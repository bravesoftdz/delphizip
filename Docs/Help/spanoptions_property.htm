SpanOptions property

Applies to

TZipMaster component

Declaration

property SpanOptions: TZMSpanOpts;

type
  TZMSpanOpts = set of TZMSpanOptsEnum;

Declaration

type
  TZMSpanOptsEnum = (spNoVolumeName, spCompatName, spWipeFiles, spTryFormat, spAnyTime, spExactName);

spNoVolumeName If True then will use numbered files on removable drives
spCompatName If True then uses names of the form name.zip + name.z01 ...
spWipeFiles If True, removes the files from the removable drive before writing.  WARNING: Deletes all files and Directories
spTryFormat If True and spWipeFiles, try to Format removable disk.  WARNING: Deletes all files and Directories
spAnyTime If True, it ignores time stamps when locating last part of set (normally time stamps must match)
spExactName If True, open the specified file instead of the last part
Description

This property is used to modify the default action of the Disk spanning operations.

If you want an option to be True, you need to add it to the Set.
