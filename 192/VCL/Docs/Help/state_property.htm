State property

Applies to

TZipMaster component

Declaration

property State: TZMStates;

type
  TZMStates = (zsDisabled, zsIdle, zsReentered, zsBusy, zsReentry);

zsDisabled the component is not Active
zsIdle the component is not busy
zsReentered the component is not busy but function just finished had reentry attempt
zsBusy the component is executing an operation
zsReentry A reentry attempt while the component is executing an operation
Description

Used to check the state of an operation in progress.

Run-time only
