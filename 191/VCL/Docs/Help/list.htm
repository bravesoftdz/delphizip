List method

Applies to

TZipMaster component

Declaration

function List: Integer;

Results
  • Returns 0 on success otherwise an ErrCode.
  • Sets ErrCode and ErrMessage
Description

The List method will read the Zip archive and refresh the contents list.

This is a manual "refresh" of the "Table of Contents".  This occurs automatically whenever a filename is assigned to ZipFileName.
