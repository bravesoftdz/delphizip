  • OnSkipped event
Applies to

TZipMaster component

Declaration

property NoSkipping: TZMSkipAborts;

type
  TZMSkipAborts = set of TZMSkipTypes;

type

Description

Sets the default action for various failures to process the requested file.

If True, that skip type will fail unless OnSkipped handler sets Err >= 0.
