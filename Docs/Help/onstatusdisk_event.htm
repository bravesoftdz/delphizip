OnStatusDisk event

  • KeepFreeOnDisk1
  • MaxVolumeSize
  • MinFreeVolumeSize
Applies to

TZipMaster component

Declaration

property OnStatusDisk: TZMStatusDiskEvent;

Type
  TZMStatusDiskEvent = Procedure(Sender: TObject; PreviousDisk: Integer; PreviousFile: String; Status: TZMZipDiskStatus; var Action: TZMDiskAction) of object;

Arguments
Sender: TObject
The component firing the event
PreviousDisk: Integer
when applicable is the disknumber of the previous disk which could be overwritten
PreviousFile: String
the filename of the zip archive.
Status: TZMZipDiskStatus
The status
var Action: TZMDiskAction
The action to take

type
  TZMZipDiskStatus = set of TZMZipDiskStatusEnum;

Type
  TZMZipDiskStatusEnum = = (zdsEmpty, zdsHasFiles, zdsPreviousDisk, zdsSameFileName, zdsNotEnoughSpace);

zdsEmpty The disk is empty and ready to use.
zdsHasFiles The disk is not empty and contains one or more file(s).
zdsPreviousDisk It's possibly a previous disk from the backup set.
zdsSameFileName The disk contains a file with the same filename as the zip archive.
zdsNotEnoughSpace There is not enough space on the disk taken into account KeepFreeOnDisk1, MaxVolumeSize and MinFreeVolumeSize.

Type
  TZMDiskAction = (zdaYesToAll, zdaOk, zdaErase, zdaReject, zdaCancel);

zdaYesToAll Continue spanning and do not ask again
zdaOk Continue diskspanning (the default).
zdaErase Erase the disk first.
zdaReject Try again.
zdaCancel Cancel the disk spanning operation.
Description
Allows responding to changes in removeable disk status during a spanned operation.
