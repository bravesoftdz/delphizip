  • About CodePages
Unit

ZMUTF8

Declaration

function PUTF8ToSafe(const raw: PAnsiChar; len: integer): AnsiString;

function UTF8ToSafe(const ustr: AnsiString; ToOEM: boolean): AnsiString;

Arguments
const raw: PAnsiChar
pointer to array of characters
len: integer
the number of AnsiChar (<0 - use to end of string)
const ustr: AnsiString
The string to convert
ToOEM: boolean
True, convert to OEM
Results
  • Returns the resulting AnsiString
Description

Converts a string of char or an Ansistring into an AnsiString escaping unsupported characters using

code page CP_OEMCP when ToOEM is true,

codepage CP_ACP when ToOEM is false.
