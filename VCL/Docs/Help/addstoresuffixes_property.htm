AddStoreSuffixes property

Applies to

TZipMaster component

Declaration

property AddStoreSuffixes: TZMAddStoreExts;

type
  TZMAddStoreExts = set of TZMAddStoreSuffixEnum;

type
  TZMAddStoreSuffixEnum = (assGIF, assPNG, assZ, assZIP, assZOO, assARC, assLZH, assARJ, assTAZ, assTGZ, assLHA, assRAR, assACE, assCAB, assGZ, assGZIP, assJAR, assEXE, assEXT, assJPG, assJPEG, ass7Zp, assMP3, assWMV, assWMA, assDVR, assAVI);

Description

AddStoreSuffixes is a Set of options to modify the default action of the Add method, files with these extensions will stored without compression.

This is because these types of files normally will not compress enough to warrant the time it takes to compress them.

  • assGIF True
  • assPNG True
  • assZ True
  • assZIP True
  • assZOO True
  • assARC True
  • assLZH True
  • assARJ True
  • assTAZ True
  • assTGZ True
  • assLHA True
  • assRAR True
  • assACE True
  • assCAB True
  • assGZ True
  • assGZIP True
  • assJAR True
  • assEXE False
  • assEXT False
  • assJPG True
  • assJPEG True
  • ass7Zp True
  • assMP3 True
  • assWMV True
  • assWMA True
  • assDVR True
  • assAVI True

// default file extensions that are best 'stored'

const
  ZMDefAddStoreSuffixes = [assGIF..assJAR, assJPG..ass7Zp, assMP3..assAVI];

Note when any type of encryption is used it is best to compress all files!
