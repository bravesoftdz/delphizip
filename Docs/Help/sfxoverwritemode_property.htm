Applies to

TZipMaster component

Declaration

property SFXOverwriteMode: TZMOvrOpts;

type
  TZMOvrOpts = (OvrConfirm, OvrAlways, OvrNever);

OvrConfirm Ask user when each file is found  (The default).
OvrAlways Always overwrite existing files.
OvrNever Never overwrite - skip those files.
Description

SFXOverWriteMode sets the the action for what the SFX program is supposed to do if it finds files that already exist.

If the option 'SFXHideOverWriteBox' is True, then this option will be used during extraction.
