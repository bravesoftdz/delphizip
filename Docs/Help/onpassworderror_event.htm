OnPasswordError event

  • Note on passwords:
  • GetExtrPassword
  • GetAddPassword
  • GetPassword
Applies to

TZipMaster component

Declaration

property OnPasswordError: TZMPasswordErrorEvent;

Type
  TZMPasswordErrorEvent = Procedure(Sender: TObject; IsZipAction: Boolean; var NewPassword: String; ForFile: TZMString; var RepeatCount: LongWord ; var Action: TMsgDlgBtn) of object;

Arguments
Sender: TObject
The component firing the event
IsZipAction: Boolean
set to true when the error is originating from an Add action else it is from an Extract action.
var NewPassword: String
The new password
const ForFile: TZMString
the path\file for which the password is needed.
var RepeatCount: Longword
first set to the PasswordReqCount number and decremented on each event call.
var Action: TMsgDlgBtn
action to take when returning from this event, possible actions are: mbOk, mbCancel, mbCancelAll or mbAbort. Default is mbOk.
Description

Occurs when adding or extracting and a password is needed but not set by the Password property and also when extracting and the given password is incorrect.

When needed you can modify RepeatCount and set it to value from 0 to 15 this will not influence PasswordReqCount.

NOTE: This event will be called at most 15 times per file even when you try to increase RepeatCount multiple times.

Example
 Procedure TMainForm.ZipBuilder1PasswordError(Sender: TObject;        
       IsZipAction: Boolean; var NewPassword: String; const ForFile: String;        
       var RepeatCount: LongWord; var Action: TMsgDlgBtn )        
 Begin        
   if IsZipAction then        
     Action := ZipMaster1.GetPassword(ZipMaster1.Info(PW_Caption),        
                                       ZipMaster1.Info(PW_MessageEnter),        
                                         DHC_AddPwrd1,        
                                       [mbOK << mbCancel],        
                                       NewPassword);        
    else        
     Action := ZipMaster1.GetPassword(ZipMaster1.Info(PW_Caption),        
                                        ZipMaster1.Info(PW_MessageEnter) +        
                                        ZipMaster1.Info(PW_ForFile) +        
                                        ForFile,        
                                        [mbOK << mbCancel << mbAll],        
                                        NewPassword );        
 End;
