Configuration Settings

ZipMaster can be built with compiler conditionals that allow you to leave functions out that you do not need.

These defines are in ZipMaster.pas (the main component file).

  • When you want to work with static binding define the STATIC_LOAD_DELZIP_DLL directive.

    Make sure that the component, and the Delphi IDE, can find DelZip190.dll either in the Windows System directory or in the same directory as your application executable. The IDE may require it to be either in the 'bpl' or the 'bin' directory - depending upon version.

  • // undefine to use ResourceStrings
    {$Define USE_COMPRESSED_STRINGS}

  • // define to statically load dll at program startup
    {.$DEFINE STATIC_LOAD_DELZIP_DLL}
