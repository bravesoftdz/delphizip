Resources

Description

  ZipMaster can use multiple languages for it's messages and can also have a copy of the dll stored in resources.

  ZipResMaker is used to select which languages are included (US English is the default).

 

  The SFXStub with its required languages can also be included in the resources. 

 

A compressed version of the dll may also be included in the resources so that it is always available.
