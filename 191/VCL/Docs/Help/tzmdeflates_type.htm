type TZMDeflates = (zmStore, zmStoreEncrypt, zmDeflate, zmDeflateEncrypt);

Description
Options for deflating/undeflating a stream.
zmStore stores the file
zmStoreEncrypt stores the file encrypted (not recomended)
zmDeflate deflate the file
zmDeflateEncrypt deflate and encrypt the file.
