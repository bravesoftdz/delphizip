Find method

Applies to

TZipMaster component

Declaration

function Find(const fspec: TZMString; var idx: Integer): TZMDirEntry;

Arguments
const fspec: TZMString
The file spec to find - can include wildcards
var idx: Integer
The index to start after
Results
  • Returns the resulting TZMDirEntry
Description

Finds the first occurance of fspec after idx.

If found sets idx to the found index and returns a read-only copy of the entry.
