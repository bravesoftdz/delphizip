ExtErrCode property

  • ErrCode
  • ErrMessage
  • Errors and Mssages
Applies to

TZipMaster component

Declaration

property FullErrCode: Cardinal;

Description

Holds a copy of the last error code sent to your program by the DLL.

The difference between ErrCode and ExtErrCode is that ErrCode is only the Message identifier whereas ExtErrCode also identifies the unit and approximate line number.

To interpret this extra information you will have to perform a logical AND and shift as follows.

Dll error code format
0FFF FFFF  LLLL LLLL   LLLL MTTT  EEEE EEEE  {31 .. 0}

F = file number (7 bits)

L = line number (12 bits)

M = message instead of error string

T = type (3 bits)

E = error/string code (8 bits)

If this property is 0 then there is no error.

Run-time only

Component error code format
0uFF FFFF  FLLL LLLL   LLLL LLuE  EEEE EEEE  {31 .. 0}

F = file number (7 bits)

L = line number (13 bits)

E = error/string code (9 bits)

u = unassigned.

If this property is 0 then there is no error.

Run-time only
