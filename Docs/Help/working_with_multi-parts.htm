Working with multi-part Zip files

  • AddOptions
  • WriteOptions
  • Working with SFX
Properties when writing
WriteOptions property
controls how to write the resulting zip
SpanOptions property
Controls how to span the resulting zip
MaxVolumeSize property
Controls the size for each part of multi-part zip
MaxVolumeSizeKb property
Controls the size for each part of multi-part zip
Floppy Disk only
KeepFreeOnAllDisks property
When writing zips spanning several floppy disks controls how much space to keep free
KeepFreeOnDisk1 property
When writing zips spanning several floppy disks controls how much extra space to keep free on the first disk
MinimumFreeVolumeSize property
When writing zips spanning several floppy disks controls how much space must be available on disk (default 65536 bytes)
OnStatusDisk property
When writing zips spanning several floppy disks responds to changes of floppy status
EncodeAs property
Controls how to store the file name (and comment) in the zip's central directory
Properties when reading
SpanOptions property
Controls how to span the resulting zip
Encoding property
Controls how to read the file name (and comment) in the zip's central directory
Description

Where possible when loading a multi-part zip it will look for the last to load but it safest to specify that part
