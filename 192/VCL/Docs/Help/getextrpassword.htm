GetExtrPassword method

  • Note on passwords:
  • GetAddPassword
  • GetPassword
Applies to

TZipMaster component

Declaration

function GetExtrPassword: String

function GetExtrPassword(var Response: TmsgDlgBtn): String

Results
  • Returns the given password.
Description

The GetExtrPassword method will ask the user for a password used while extracting files.

Prompt user for a password.  The password will only be asked once. It will be saved in the Password property and it will also be returned as result.

The use of this method is not required. If you want to make your own password prompt dialog box, you can just put the password into the password property yourself. Also, you can take the easiest route by leaving the password property blank, and letting the component prompt the user for the password.

The overloaded version also gives the user's response.
