Applies to

TZipMaster component

Declaration

property SFXMessage: TZMString;

Description

You can specify a message that will be displayed before the main extraction dialog pops up.

You can also specify the type of the message box:
Prefix Type explanation
  Standard  The messagebox shows an 'OK' button. The User cannot cancel starting the SFX application.
1| Information  The messagebox shows the buttons 'OK' and 'Cancel'. if the User presses 'Cancel', sfx will stop.
2| Confirmation  The messagebox shows the buttons 'Yes' and 'No'. if the User presses 'No', sfx will stop.

 

NOTE: When the first character of the string = '|' then all following chars will be included in the message.
