GetAddPassword method

  • Note on passwords:
  • GetExtrPassword
  • GetPassword
Applies to

TZipMaster component

Declaration

function GetAddPassword: String

function GetAddPassword(var Response: TmsgDlgBtn): String

Results
  • Returns the password when verified otherwise an empty string.
Description

The GetAddPassword method will ask the user for a password used while adding files.

Prompt user for a password. The password will be asked twice - the second time to verify it.

If both of them match, it will be saved in the Password property and it will also be returned as result, or else the Password property will not change and the result is an empty string.

The use of this method is not required. If you want to make your own password prompt dialog box, you can just put the password into the password property yourself. Also, you can take the easiest route by leaving the password property blank, and letting the component prompt the user for the password.

The overloaded version also gives the user's response.
