SFXOffset property

Applies to

TZipMaster component

Declaration

property SFXOffset: Integer;

Description

SFXOffset is the size of the self-extracting code at the beginning of the zipfile.

SFXOffset is valid after a zip file's contents have been read successfully (which happens when you assign something to ZipFileName). When using non SFX zip files or when using diskspanning this will be zero.

Run-time only

Read-only
