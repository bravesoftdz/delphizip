OnCopyZippedOverwrite event

  • CopyZippedFiles
  • AddZippedFiles
Applies to

TZipMaster component

Declaration

property OnCopyZippedOverwrite: TZMCopyZippedOverwriteEvent;

Type
  TZMCopyZippedOverwriteEvent = procedure(Sender: TObject; src, dst: TZMDirEntry; var DoOverwrite: Boolean) of object;

Arguments
Sender: TObject
This component
src: TZMDirEntry
The entry being copied
dst: TZMDirEntry
The existing entry
var DoOverwrite: Boolean
Set True to overwrite existing entry (default: False)
Description

Occurs when during the execution of the function CopyZippedFiles or AddZippedFiles it is necessary to know if a file in the destination archive may be overwritten. It is possible that the event does NOT occur because you have let the function CopyZippedFiles know that it is always or never Ok to overwrite.

You can choose to overwrite the file dst.FileName by setting DoOverwrite to True. The default for DoOverwrite is False.
