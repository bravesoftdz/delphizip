ConfirmErase property

Applies to

TZipMaster component

Declaration

property ConfirmErase: Boolean;

Description

ConfirmErase can set ZipMaster to ask for confirmation before each disk will be erased when the SpanOption spTryFormat is set.

If you set this to False, you will not be asked for confirmation before a disk is erased. If True and the answer is No, the disk in question will be used as it is. If there is not enough space available you will be prompted for a new one.

Default value = True
