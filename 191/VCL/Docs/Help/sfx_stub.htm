SFX Stub

The SFX stub is the executable self extractor that does the actual extraction of the zip files.

 

The source code for the raw stub is distributed in the SFX\  folder.

Depending upon which compiler version is used the stub will either be -

D2009 or later - ZMSFXU191.exe, Unicode that will only run on Windows XP or later.

D2007 or earlier - ZMSFX191.exe, Ansi that will run on most versions of Windows but may have trouble with file names with non-US characters.

 

These raw stubs have to be prepared using the tool ZipResMaker.exe which appends compressed tables of the required strings and messages in various languages.

ZipResMaker.exe also prepares the stubs as resources that can be attached to the application so that external files are not required.

 

Prepared stubs are distributed in the \Res  folder as -

ZMSFXU191.bin (Unicode).

ZMSFX191.bin (Ansi).

ZMRes191_sfxu.res (Unicode) **.

ZMRes191_sfx.res (Ansi) **.

 

** either ZMRes191_sfxu.res or ZMRes191_sfx.res may be linked but not both at the same time!
