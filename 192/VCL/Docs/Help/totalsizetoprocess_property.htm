TotalSizeToProcess property

Applies to

TZipMaster component

Declaration

property TotalSizeToProcess: Int64;

Description

Total uncompressed file size of all files going to be processed while doing an active operation.

It is set in the OnProgress event with event type: TZMTotalSize2Process.

Run-time only

Read-only
