IsSameFile function

Unit

ZMUtils

Declaration

function IsSameFile(const FName1, FName2: String): Boolean;

Arguments
const FName1: String
a file name
const FName2: String
the other file name
Results
  • Returns True if same otherwise false
Description

Returns true if both Fname1 and Fname2 are equal after expanding to full path.

If only their filenames are different then we try to find both files, if they are found the file attributes are compared, if these are equal they are assumed to be the same file.
