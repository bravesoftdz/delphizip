UseDirOnlyEntries property

Applies to

TZipMaster component

Declaration

property UseDirOnlyEntries: Boolean;

Description

UseDirOnlyEntries can be set to include directory name entries in the ZipDirEntry records.

UseDirOnlyEntries can be set to True to include directories in the ZipDirEntry array, default is False
