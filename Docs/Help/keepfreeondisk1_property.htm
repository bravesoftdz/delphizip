KeepFreeOnDisk1 property

Applies to

TZipMaster component

Declaration

property KeepFreeOnDisk1: Cardinal;

Description

KeepFreeOnDisk1 is used to keep some space available on the first disk of a spanned set.

Holds a value in bytes and will use this value to keep bytes free on the first disk of a set of disks.

For example to leave space for a setup program.

By default this is set to zero.
