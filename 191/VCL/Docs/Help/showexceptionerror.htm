ShowExceptionError method

Applies to

TZipMaster component

Declaration

procedure ShowExceptionError(const ZMExcept: EZipMaster);

Arguments
const ZMExcept: EZipMaster
the exception
Description

If Unattended is True, sends the exception message through the OnMessage event, otherwise it is displayed in a MessageBox.

EZipMaster, derived from Exception, is used for exceptions throughout the component code.
