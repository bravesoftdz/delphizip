TempDir property

Applies to

TZipMaster component

Declaration

property TempDir: String

Description

If TempDir is set to a drive/directory then this full path will be used.

While using Diskspanning:

This can be useful when spanning large archives.

If this property is not set then the used path is the system default temporary directory. This is either the path set with the TEMPDIR or TMPDIR environment variable or if both are not available the current directory on the current drive.

While using the DLL:

If this property is not set then the used path is the same directory as where the zipfile is located.
