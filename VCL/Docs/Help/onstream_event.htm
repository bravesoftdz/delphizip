OnStream event

  • TZMPipe
  • TZMPipeList
Applies to

TZipMaster component

Declaration

property OnStream: TZMStreamEvent;

type
  TZMStreamEvent = procedure(Sender: TObject; opr: TZMStreamOp; snumber: integer;  var strm: TStream; var stat: TZMSStats; var done: Boolean) of object;

Arguments
Sender: TObject
The component firing the event
opr: TZMStreamOp
The operation being performed
snumber: integer
The number assigned the stream
var strm: TStream
Reference to the stream
var stat: TZMSStats
Used to identify stream properties
var done: Boolean
Set true when done

type
  TZMStreamOp = (zsoIdentify, zsoOpen, zsoClose);

Type

   TZMSStats = packed record

  • size: Int64;
  • Date: Cardinal;
  • Attrs: Cardinal;

  end;

Description

It can be used to Create/Open, Destroy/Close or Identify a stream.

If done is set it will override the default handling of stream operations.
