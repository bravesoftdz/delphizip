OnSkipped event

  • NoSkipping
  • TZMSkipAborts
  • TZMSkipTypes
Applies to

TZipMaster component

Declaration

property OnSkipped: TZMSkippedEvent;

Type
  TZMSkippedEvent = procedure(Sender: TObject; const ForFile: TZMString; SkipType: TZMSkipTypes; var ExtError: Integer) of object;

Arguments
Sender: TObject
The component firing the event
const ForFile: TZMString
The file that will be skipped
SkipType: TZMSkipTypes
Reason for skipping
var ExtError: Integer
  • On entry ExtError will be the >= 0 if NoSkipping for that type is false
  • On return if < 0 the operation will abort.
Description

Fired when an operation fails to process an instruction or file.

On entry ExtError will be the >= 0 if NoSkipping for that type is false

If ExtError < 0 upon return from a handler the operation will abort.
