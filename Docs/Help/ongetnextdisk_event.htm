OnGetNextDisk event

Applies to

TZipMaster component

Declaration

property OnGetNextDisk: TZMGetNextDiskEvent;

Type
  TZMGetNextDiskEvent = procedure(Sender: TObject; DiskSeqNo, DiskTotal: Integer; Drive: String, var Cancel: Boolean) of object;

Arguments
Sender: TObject
This component
DiskSeqNo: Integer;
the sequence number of the disk needed or is zero when it concerns a diskspan read action.
DiskTotal: Integer;
DiskTotal the total number of needed disks. This is an estimate based on some known values or it is zero when the number could not be calculated.
Drive: String
the drive name, e.g. 'A:'.
var Cancel: Boolean
Cancel, can be set to True to cancel the spanning operation, default False.
Description

Occurs when reading or writing multi-disk sets.
