EraseFile method

Applies to

TZipMaster component

Declaration

function EraseFile(const FName: String; How: TZMDeleteOpts): Integer;

Arguments
const FName: String
?
How: TZMDeleteOpts
The method to use
Results
  • Returns 0 on success otherwise a negative value equal to - Error Code.
Description

EraseFile is like DeleteFile, but depending on the value of 'How' the specified file will be deleted completely or will be put into the recycle bin. When just a filename is specified the drive and directory will be taken from the current drive and directory.

EraseFile is also used internally by the component functions ConvertZip, ConvertSFX, CopyFile, CopyZippedFiles and ReadSpan but then uses the property HowToDelete as How value.
