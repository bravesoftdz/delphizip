OnStateChange event

Applies to

TZipMaster component

Declaration

property
  OnStateChange: TZMStateChange;

type
  TZMStateChange = procedure(Sender: TObject; state: TZMStates; var NoCursor: boolean); of object;

Arguments
Sender: TObject
The component firing the event
state: TZMStates;
The new operational state
var NoCursor: boolean
  • true the current cursor will not be changed.
  • false the cursor will reflect the Active state.
Description

Allows handling changes in the component state eg. to block button clicks when busy.

If Active the Waitcursor has started, going not Active stops the Waitcursor.
