Unit

ZMUTF8

Declaration

function UTF8IsSafe(const ustr: AnsiString; ToOEM: boolean): Boolean;

Arguments
const ustr: AnsiString
The string to test
ToOEM: boolean
True to restrict to those characters safe in OEM page
Results
  • Returns true if all characters in ustr are supported
Description

Tests a string to check that all characters are supported using

  • code page CP_OEMCP when ToOEM is true
  • codepage CP_ACP when ToOEM is false.
