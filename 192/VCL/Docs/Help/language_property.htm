Applies to

TZipMaster component

Declaration

property Language: string

Description

Controls the language strings used for ErrMessage.

Sets and returns the current language.

usage:

Language := 'FR'; // set to French
Language := '*'; // set to current user language

Notes

if a language cannot be found the default 'US: English' will be used.

Languages other than the default US require the resources to be linked to the application
