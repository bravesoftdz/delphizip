TZMString type

Unit

ZipMstr

Declaration

Because the newer versions of Delphi now use UnicodeString instead of AnsiString the actual string types used by TZipMaster will depend upon the Delphi version.

type
  TZMString = UnicodeString;		 D2009 or later (Unicode)
type
  TZMString = AnsiString;		 D2007 or earlier (Ansi)
Description

This string type is used for file entry names to enable handling of international names.

On Unicode versions (D2009 or later) this is a standard String.

On earlier versions its contents will be either AnsiString or, if UseUTF8 is True, UTF8.
