Applies to

TZipMaster component

Declaration

property DLL_Load: Boolean;

Description

Controls the loading/unloading of dynamically loaded DLL

Set True to load the DLL or False to unload it.
if Active is false the request to load will handled after Active is True!

Read it to check if the DLL is actually loaded.

Notes:

Clearing it while an operation is running will unload when finished.

Ignored if compiled with STATIC_LOAD_DELZIP_DLL defined.
