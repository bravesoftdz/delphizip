Results
  • Returns 0 on success otherwise a negative value equal to - Error Code.
  • Sets ErrCode and ErrMessage
Description

The WriteSpan method can be used to copy an existing zip archive file to one or more disks as a spanned archive.

With WriteSpan it is possible to make a spanned archive on (floppy) disk from a standard zip file.

InFileName is the name of the zipfile that must be converted.

OutFileName is the drive, path(optional) and file name of the destination file.

NOTE: If 'OutFileName' is on a fixed disk the input file will be split into parts with size MaxVolumeSize A file part on harddisk will look like: OutFileNameXXX.zip where XXX is a number.

It is possible to copy the files to diskette when the filesize is <= diskette size.

or to reassemble them without diskettes after sending them over the internet for use with pkzip or winzip.
