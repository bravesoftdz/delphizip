StrToWideEx function

Unit

ZMUTF8

Declaration

function StrToWideEx(const astr: AnsiString; cp: cardinal; len: integer): TZMWideString;

Arguments
const astr: AnsiString
The string to convert
cp: cardinal
The code page to use (0 = default)
len: integer
The number of AnsiChars to convert (-1 for all)
Results
  • Returns the resulting TZMWideString
Description

On Unicode versions (CB2009 or later) Converts an AnsiString into a standard UnicodeString.

On earlier versions Converts an AnsiString into a WideString (UTF16).
