Trace property

  • Verbose
  • Unattended
  • OnMessage
Applies to

TZipMaster component

Declaration

property Trace: Boolean;

Description

Trace is an option to get extended messages during active operations.

Similar to Verbose, except that this one is aimed at developers. 

It lets you trace the execution of the code. Helps you locate possible bugs, and helps you understand why something is happening a certain way.

NOTE: Text messages for the Trace come to you through the OnMessage event.
