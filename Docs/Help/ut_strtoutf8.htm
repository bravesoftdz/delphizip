StrToUTF8 function

Unit

ZMUTF8

Declaration

function StrToUTF8(const ustr: String): UTF8String;

Arguments
const ustr: String
the string to convert
Results
  • Returns the resulting UTF8String
Description

Converts a String into an UTF8String

On Unicode versions (CB2009 or later) String is a standard UnicodeString.

On earlier versions String is an AnsiString
