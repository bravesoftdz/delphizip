Applies to

TZipMaster component

Declaration

property LanguageInfo[Idx: Integer;info: Cardinal]: String

Description

returns information about a language

Idx values

  • < 0   - default US
  •    0   - current language
  • > 0   - Available language number (requesting beyond available number of languages returns '><')

Info values

  • $FFFFFFFF - returns language identifier of language
  • 0   - returns name of language (i.e. 'SP'  = Spanish)
  • Windows LOCALE_ prefixed values - returns that information from Windows.

Run-time only

Read-only

Example
 // to list all available languages
 for i := 0 to 50 do
 begin
    s := LanguageInfo[ i, LOCALE_SENGLANGUAGE ];
    if s = '><' then
       break;
    Memo1.Lines.Add( s );
 end;
