TZMWideString type

Unit

ZipMstr

Declaration

Because the newer versions of Delphi now use UnicodeString instead of AnsiString the actual string types used by TZipMaster will depend upon the Delphi version.

type
  TZMWideString = UnicodeString;		 D2009 or later (Unicode)
type
  TZMWideString = WideString;		 D2007 or earlier (Ansi)
Description

A string of Wide characters (UTF16).
