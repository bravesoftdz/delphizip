OnCheckTerminate event

Applies to

TZipMaster component

Declaration

property OnCheckTerminate: TZMCheckTerminateEvent;

Type
  TZMCheckTerminateEvent = Procedure(Sender: TObject; Var abort: boolean) Of Object;

Arguments
Sender: TObject
This component
var abort: Boolean
set True to terminate
Description

used for Multi-threaded applications

Stops message processing from calling ProcessMessages (not allowed outside main thread).

Allows checking for Terminate

Example
 // stops calls to ProcessMessages & checks for Terminated
  procedure TZipThread.CheckTerminate(Sender: TObject; var abort: boolean);     
  begin
    if Terminated then
      abort := true;
  end;
