MakeTempFileName method

Applies to

TZipMaster component

Declaration

function MakeTempFileName(const Prefix, Extension: String): String

Arguments
const Prefix: String
The requested prefix for the name
const Extension: String
the extension to give it
Results
  • Returns the resulting file name and path
Description

The MakeTempFileName method constructs a filename of the form 'C:\....\zipxxxx.zip' where xxxx is a unique number.

Prefix is per default: 'zip' = and Extension is '.zip'.

MakeTempFileName constructs a temporarily filename, that on the moment of construction does not have an equivalent on disk.

The actual file created on disk will be removed before returning the filename.

If the TempDir is defined then this path will be used to construct the filename else the TMP or TEMP environment variable will be used, if both TMP or TEMP are not defined then the current drive and directory will used as path.
