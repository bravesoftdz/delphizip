OnExtractOverwrite event

Applies to

TZipMaster component

Declaration

property OnExtractOverwrite: TZMExtractOverwriteEvent;

Type
  TZMExtractOverwriteEvent = procedure(Sender: TObject; const ForFile: TZMString; IsOlder: Boolean; var DoOverwrite: Boolean; DirIndex: Integer) of object;

Arguments
Sender: TObject
This component
const ForFile: TZMString
the name of the file that will be overwritten.
IsOlder: Boolean
IsOlder indicates if the existing file, on disk, is older.
var DoOverwrite: Boolean
DoOverwrite can be set to True when you want the file to be overwritten.
DirIndex: Integer
DirIndex is the index of ForFile in the internal list. With this you have full access to all available archive file information for this file.
Description

Occurs during Extract operation before overwriting an existing file.
