PasswordReqCount property

Applies to

TZipMaster component

Declaration

property PasswordReqCount: Longword;

Description

Password Request Count is used for setting the total number of times a password is requested per file in the ziparchive (by calling OnPasswordError). The maximum is 15 and the minimum = 0.

Setting it to 0 will make sure that a password is never requested, effectively skipping all encrypted files.

NOTE: Setting this property while extracting is in progress does not have any effect until the next call to Extract.
