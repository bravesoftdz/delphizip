Verbose property

  • Trace
  • Unattended
  • OnMessage
Applies to

TZipMaster component

Declaration

property Verbose: Boolean;

Description

Verbose is an option to get more messages from operations.

If True, ask for the maximum amount of "possibly important" information from the operation.

The informational messages are delivered to your program via the OnMessage event, and the ErrCode and ErrMessage properties. 

This is primarily used to determine how much info you want to show your "end-users" - developers can use the Trace property to get additional information.
