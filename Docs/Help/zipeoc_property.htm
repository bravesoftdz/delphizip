ZipEOC property

Applies to

TZipMaster component

Declaration

property ZipEOC: Int64;

Description

ZipEOC contains the location measured from the start of the zip file of the End-Of-Central-Dir.

ZipEOC is valid after a zip file's contents have been read successfully (which happens when you assign something to ZipFileName).

Run-time only

Read-only
