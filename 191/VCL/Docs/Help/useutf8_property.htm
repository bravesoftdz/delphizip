UseUTF8 property

Applies to

TZipMaster component

Declaration

property UseUTF8: Boolean;

Description

Delphi versions before D2009 only.

Early Delphi versions do not natively support Unicode so to support international zip entry strings UseUTF8 can be set.

This means that TZMString will be assumed to be UTF8.
