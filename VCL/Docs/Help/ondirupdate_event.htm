OnDirUpdate event

Applies to

TZipMaster component

Declaration

property OnDirUpdate: TNotifyEvent;

Description

OnDirUpdate event occurs when ZipMaster has refreshed the contents list.

Occurs immediately after ZipMaster reads the central directory.

This is your cue to update the screen with the new contents.
