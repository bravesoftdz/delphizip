Applies to

TZipMaster component

Declaration

property FSpecArgsExcl: TStrings;

Description

Stringlist containing all the file specifications to be excluded for the following methods

  • Add
  • AddZippedFiles
  • Delete
  • ChangeFileDetails
  • CopyZippedFiles
  • Delete
  • ForEach
  • MergeZippedFiles

Every entry can contain MS-DOS wildcards.

You can specify either the MS-DOS backslash path symbol, or the Unix path separator: /). They are treated exactly the same.

When using full path names in FSpecArgs you must also use full path names in FSpecArgsExcl or appropriate wildcards like '*.tmp', '*test.*', this is important when the Add option AddRecurseDirs is set.

Multiple filespecs can be specified by delimiting with |

NOTE: Before D2009,  if UseUTF8 is True the strings must be UTF8.

Example
 Add entries directly to the FSpecArgsExcl property:        
 ZipMaster1.FspecArgsExcl.Add( 'C:\Builder\BIN\Builder.EXE' );        
 ZipMaster1.FspecArgsExcl.Add( 'C:\WINDOWS\*.TMP' );        
 ZipMaster1.FspecArgsExcl.Add( '*.TMP' );        
          
 Take the filespecs from a StringList, just assign them all over to ZipMaster1.        
 ZipMaster1.FspecArgsExcl.Assign( StringList1 );        
          
 Take the filespecs from a ListBox, just assign them all over to ZipMaster1.        
 ZipMaster1.FspecArgsExcl.Assign( ListBox1.Items );
