AddCompLevel property

Applies to

TZipMaster component

Declaration

property AddCompLevel: Integer;

Description

Compression Level.

Range 0 - 9, where 9 is the tightest compression.  2 or 3 is a good trade-off if you need more speed. Level 0 will just store files without compression.

I recommend leaving this at 9 in most cases.
