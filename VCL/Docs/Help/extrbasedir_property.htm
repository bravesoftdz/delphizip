ExtrBaseDir property

Applies to

TZipMaster component

Declaration

property ExtrBaseDir: String

Description

ExtraBaseDir is base directory used for extracting a Zip archive.

This base directory applies only to 'Extract' operations.   

If you don't specify a value for this property, then the directory of the ZipFile itself will be the base directory for extractions.
