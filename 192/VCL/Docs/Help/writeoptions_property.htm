Declaration

property WriteOptions: TZMWriteOpts;

type
  TZMWriteOpts = set of TZMWriteOptsEnum;

type
  TZMWriteOptsEnum = (zwoDiskSpan, zwoZipTime, zwoForceDest, zwoSafe);

zwoDiskSpanWrite as multi-part zip (using SpanOptions)
zwoZipTimeSet the zip time stamp to the latest contained file
zwoForceDestTry to force that the destination folder exists
zwoSafeWrite a temporary zip first then replace the original file
Description

Controls how a zip file will be written.
