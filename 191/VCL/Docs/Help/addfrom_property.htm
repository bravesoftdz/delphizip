AddFrom property

  • AddOptions
  • FSpecArgs
Applies to

TZipMaster component

Declaration

property AddFrom: TDateTime;

Description

AddFrom can be used to set the Date for filtering files to add to a Zip.

The date used when the AddOption - AddFromDate is set, files dated on or after this date can be selected.
