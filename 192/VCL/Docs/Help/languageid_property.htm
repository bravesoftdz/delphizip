Applies to

TZipMaster component

Declaration

property LanguageId: Cardinal

Description

Indicates what language in the resource file has to be loaded.

The language can be indicated with the numeric value of the LCID.

If the identified language is not found the 'primary' language will be searched and used if found.

usage:

LanguageId := 1036; // set to French
LanguageId := $040C; // set to French
LanguageId := 1024; // set to current user language
LanguageId := 12; // set to any language it can find that has French as 'primary' language

Notes

if a language cannot be found the default 'US: English' will be used.

Languages other than the default US require the resources to be linked to the application
