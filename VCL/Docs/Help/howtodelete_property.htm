HowToDelete property

Applies to

TZipMaster component

Declaration

property HowToDelete: TZMDeleteOpts;

type TZMDeleteOpts = (htdFinal, htdAllowUndo);

htdFinal Perform a DeleteFile.
htdAllowUndo Move the old file into the recycle bin (The default).
Description

HowToDelete specifies on how EraseFile is going to delete a file.
